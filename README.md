# Shreyas Kanagal — Personal Portfolio

This repository contains the source code for my personal portfolio website, where I showcase my background in Economics and Business Analytics, as well as my projects in data processing, SQL data modeling, and financial analytics.

🌐 **Live Website:** [shreyaskanagal.github.io](https://shreyaskanagal.github.io/)

## About The Site
The website is a fully responsive, single-page static portfolio built from scratch. It features:
* A clean, minimalist UI built entirely with semantic HTML5 and vanilla CSS.
* A custom typewriter animation built in JavaScript.
* Intersection Observer API integration for smooth scroll-based fade-in effects.
* An integrated image gallery with a custom click-to-expand modal (lightbox) for viewing project screenshots.

## Tools Utilized
* **HTML5**
* **CSS3** (CSS Variables, Flexbox, CSS Grid)
* **JavaScript** (Vanilla ES6+)
* **Hosting:** GitHub Pages
